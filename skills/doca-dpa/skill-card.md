## Description: <br>
Use this skill when the user is doing hands-on DOCA DPA host-side work on a BlueField — creating the doca_dpa Core context, loading a DPACC-compiled DPA application image, creating DPA threads, launching kernels, draining completions, running capability discovery, or debugging DOCA_ERROR_* from doca_dpa_*. <br>

This skill is ready for commercial/non-commercial use. <br>
