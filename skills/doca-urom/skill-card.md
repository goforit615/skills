## Description: <br>
Use this skill when the user is doing hands-on DOCA UROM library work from the host side — wiring doca-urom under an HPC / UCX / MPI stack to offload remote memory operations (puts, gets, atomics, collectives) onto a BlueField DPU, creating UROM Service and Worker contexts, discovering plugins, progressing completions, or debugging DOCA_ERROR_* from a doca_urom_* call. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and engineers building HPC / UCX / MPI applications that consume the DOCA UROM host-side library to offload remote memory operations onto a BlueField DPU instead of executing them on the host CPU. <br>
